# Mini Campaign Manager - Claude Code Context

## Project Overview

This is a **full-stack Mini Campaign Manager** application - a MarTech tool for creating, managing, and tracking email campaigns. The project is built as a monorepo using yarn workspaces.

**Tech Stack:**
- Backend: Node.js, Express, PostgreSQL, Sequelize ORM
- Frontend: React 18+, TypeScript, Vite, React Query, Zustand/Redux
- Infrastructure: Docker Compose, Yarn Workspaces
